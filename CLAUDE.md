# CLAUDE.md

Guidance for working in this repository.

## Language conventions

- **All code comments MUST be written in English.** This applies to every
  comment in the codebase (TypeScript, config files like `wrangler.jsonc`,
  scripts, etc.). Do not write comments in Spanish.
- **User-facing UI strings stay in Spanish** (the app's audience is Spanish).
  Labels, buttons, placeholders, error messages shown to the user, etc.
- Documentation (`README.md`, this file) is written in English.

## Project

Internal, private web app for **Terapeutica Spine SL** to list, search and
download patient image folders stored in Cloudflare R2, behind Clerk auth.
See `README.md` for stack, deployment, and configuration details.
